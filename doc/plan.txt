General objectives:

Create a interface that allows the user to solve traditional sudoku
problems and implement strategies to solve harder games.


Specific objectives:

- Study and divide the problem in a intuitive way, creating
    intuitive auxiliar classes.
- Implement a intuitive interface to traditional sudoku games.
- Implement a intuitive interface to auxiliar classes, such as
    Groups and Cells.
- Implement a library portable and easy to use.
- Store files in HDF5 format, easy to read and visualize.
- Be simple, consistent and efficient.
